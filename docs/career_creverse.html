1 (주)크레버스

회사 소개



주식회사 크레버스(구: 청담러닝)는 대한민국의 교육기업으로, 2002년 6월에 설립되었다. 주축사업은 학원, 온라인 강의, 스마트 러닝이다.
2008년 6월 코스닥에 상장했다.
2022년 3월 청담러닝과 씨엠에스에듀 합병 후 영어 교육, 수학 교육, 사고력 개발 등 포괄적인 교육 프로그램과 교재를 개발하고 제공하고 있다.

주요 사업 브랜드

  • 청담어학원
  • 에이프릴어학원
  • i-Garten
  • CMS영재교육센터

2.5 Bouncy 이슈 처리(2022.01 ~ 2022.06)

프로젝트 개요

외주사가 개발한 Bouncy 프로젝트 이슈 발생 및 Jira에 등록된 이슈 중 수정 가능한 이슈를 확인하여 처리

인원: 2명

프로젝트 진행

  • 프로젝트 기획안, 디자인과 다른 부분 수정
  • Agora Engine의 크로마키 영상 녹화 관련 녹화 끊김 현상을 학생이 생성한 크로마키 이미지의 해상도 및 품질 조절로 개선
  • Photon을 이용한 강사, 학생 데이터 전송 관련 기존 강사 타이핑마다 데이터 전송에서 입력 후 1초 동안 입력이 없을 경우 전송으로 수정
  • 학생 해상도 관련 위치 동기화가 필요한 UI를 기존 anchoredPosition에서 transform.position으로 변경 후 상대 좌표를 이용해 위치 동기화 수정
  • 강사 듀얼 모니터 수업으로 인해 클릭이 안되는 문제가 발생하는 부분을 Display.RelativeMouseAt()을 사용하여 수정

프로젝트 성과

  • 약 250건의 이슈 중 125건의 이슈 수정으로 인한 프로젝트 안정화
  • Unity UI 컴포넌트를 활용하여 스크립트 내 동적 레이아웃 변경 코드 제거 및 불필요한 스크립트 정리

기여도: 50%

기술 스택

Unity, C#, UGUI, Agora Engine, Photon

비고

  • Jira 이슈 수정 내역, Bouncy Photon 개선 내역을 구글 시트로 관리하여 개선 진행 상황 확인
