1 개요

게임 설명

캔디를 떨어뜨려서 무지개 캔디를 만드는 게임이다.
스테이지가 높아질수록 캔디를 떨어뜨릴 수 있는 공간이 점점 줄어든다.
2 특징

캔디 종류

총 10종류의 캔디가 있다. 순서가 높아질수록 크기도 커진다.

클리어 조건

캔디를 떨어뜨려서 무지개 캔디를 만들면 클리어 할 수 있다.

스테이지

2024년 08월 기준 총 6개의 스테이지가 있다.
3 개발

개발 동기

기존 장르의 게임인 수박 게임을 플레이 하는 중에 좀 더 재밌는 상황에서 플레이 하면 재밌을 것 같다는 생각으로 다양한 스테이지가 있는 수박 게임 개발을 시작했다.

주요 기능

  • CircleCollider2D, Rigidbody2D

기타

  • Candy, UI 등 리소스는 Figma로 직접 제작
  • Sound 리소스는 Unity Asset Store 활용
