1 개요

게임 설명

네크로맨서가 되어 전투에서 쓰러진 적들의 시체를 마법으로 부활시켜 아군으로 만들 수 있다.
시체들을 다시 움직이게 만들어 군대를 만들고 적들을 섬멸해야 하는 어드벤처 장르의 게임이다.
2 트레일러
3 특징

전투

일반적인 게임이랑은 다르게 적을 죽여도 사라지지 않는다. 일정 시간이 지나거나 맵을 이탈할 경우 사라진다.

스킬

소생: 쓰러진 적을 마법으로 되살리고, 아군으로 만들 수 있다.



소환: Chapter 2를 클리어하면 용 소환 스킬을 사용할 수 있다.

4 개발

개발 동기

영화, 드라마, 소설 등 네크로맨서는 죽은 시체를 되살려 아군으로 만드는 캐릭터로 나오지만, 대부분의 게임 속 네크로맨서는 해골 소환사의 느낌을 많이 받았다. 몬스터를 죽여도 오브젝트가 사라지지 않고 스킬을 사용하면 적이 되살아나며 플레이어의 편이 되는 네크로맨서 게임을 직접 플레이 해보고 싶어서 개발했다.

주요 기능

  • UI 캐릭터 컨트롤러
  • NavMeshAgent를 이용한 캐릭터 이동
  • Unity Navigation을 이용한 자동 길찾기 기능

기타

  • 캐릭터, 이펙트, UI, Sound 등 리소스는 Unity Asset Store 활용
